Cobalt. Web Medical Practice Management System

Extremely complete, easy to use, robust Web Radiologic Information System and Medical Practice Management Sofware

Sep 3, 2012 1 min read

Developed using Ruby on Rails, I managed to create a medical management web application deployable locally or on the cloud from any vendor. The software was able to manage doctor’s appointments, patients medical records, invoices, treatments, medications and it had a dashboard for showing center performance. Codename: Cobalt

It was a very complete piece of software, robust, fast and easy to use. Key features:

  • Patient management module
  • Appointments
  • Reports
  • Invoices
  • OsiriX integrated workflow
  • Integrated with Osirix for image review and other external DICOM Viewers
  • Built-in DICOM Worklist server
  • and more…
