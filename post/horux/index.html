HoruX. Mac OS X Medical Management Application

Beautiful complete and easy to use Radiologic Information System

Jun 20, 2010 1 min read

HoruX is (was) a Mac OS X Medical Management desktop application with PostgreSQL database server. I developed it as a Radiologic Information System (RIS) compatible with DICOM servers (know as PACS) and integrated with OsiriX (DICOM Medical Image Viewer). Codename: HoruX.

It was a very complete piece of software, robust, fast and easy to use. Key features:

  • Patient management module
  • Appointments
  • Reports
  • Invoices
  • OsiriX integrated workflow
  • Voice dictation and recorder
  • Report encrypted email sending
  • Built-in DICOM Worklist server
  • and more…
